known Issues

Resetting password

Currently, there is a small bug with this feature. It's working, but when you set a new password, you might get stuck on the loading page. Don't worry, just quit the app and launch it again; it will work.

Latency in user's lists

If you have a high number of games in your lists (especially when browsing in "all"), it might become a bit tedious and buggy. We're working on it.

Currency isn't displaying correctly

There is an issue with how your currency is displayed. It's just a visual bug, and we're on it.

Games that are released annually (FIFA, NBA2K...)

Some of these games use the same name for the process. For example, if you played Football Manager 2023 and then bought Football Manager 2024 and are playing it, FM23 might be detected as the game played. As a temporary fix, go to the FM23 game page and click on the "Delete game path" button right under the time spent section. This will allow you to bind the Football Manager process to another game.
